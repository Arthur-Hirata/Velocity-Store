## 📖 Sobre o projeto

Este projeto foi minha **primeira experiência desenvolvendo uma aplicação Full Stack**.

A proposta foi criar um **E-commerce de peças para computadores**, implementando autenticação de usuários, gerenciamento de produtos, banco de dados e um painel administrativo completo.

# ✨ Funcionalidades

## 👤 Área do Usuário

- Cadastro de usuários
- Login e autenticação
- Conta individual para cada usuário
- Lista de compras personalizada
- Visualização dos produtos
- Carrinho de compras



## 🔐 Segurança

- Senhas armazenadas utilizando **Hash**
- Autenticação de usuários
- Controle de acesso para administradores


## 🛠 Painel Administrativo

- Dashboard administrativo
- Visualização das últimas vendas
- Cálculo de lucro
- Listagem de usuários
- Listagem de produtos
- Cadastro de novos produtos
- Edição de produtos
- Remoção de produtos
- Exclusão de usuários

Durante o desenvolvimento deste projeto pratiquei conceitos como:

# 🎯 Objetivo

Este projeto marcou meu primeiro contato com o desenvolvimento Back-end utilizando Python e SQL.

Nele pude compreender como uma aplicação web funciona de ponta a ponta, desde a interface do usuário até a persistência de dados em um banco de dados relacional.

Além disso, implementei autenticação, APIs, operações CRUD, controle de permissões e um painel administrativo, consolidando fundamentos importantes do desenvolvimento web.
